## 1.2 JS基础面试题

**(1) 什么是闭包？**

(犀牛书)闭包： 函数变量可以保存在函数作用域内
```js
function foo1() {
  var n = 2;
}
```

(高程)闭包：闭包是指有权访问另一个函数作用域中变量的函数（函数没导出）
```js
function foo2() {
  var n = 0;
  function innerFoo() {
    console.log(n);
  }
  innerFoo();
}
foo2();
```

(你不知道的js)当函数可以记住或者访问所在的词法作用域时，就产生了闭包，即使函数是在当前的词法作用域之外
```js
function foo3() {
  var n = 1;
  return function innerFoo() {
    console.log(n);
  };
}
foo3()();
```

(MDN)一个函数和对其周围状态的引用捆绑在一起，这样的组合就是closure

总结：当函数的执行，导致函数被定义
